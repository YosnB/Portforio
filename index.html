自己紹介 (About Me)

スキルと関心

大学では、Webデザイン、データ分析、情報システム設計を主に学んでいます。特に**ユーザー目線での課題発見と改善プロセス**に強い関心があります。

  • **技術スキル:** HTML, CSS, JavaScript (初級), Python (データ処理)
  • **得意分野:** 情報の整理・可視化、論理的な問題解決、関係者間の調整

学生時代に力を入れたこと (ガクチカ)

高校時代、ICT委員会委員長として、学校行事のWebサイト作成などのIT活動を主導しました。

特に力を入れたのは、全生徒に配布されていた**iPadの利用規制改革**です。当時、標準アプリも使えないほどの過剰な制限があり、生徒の利便性を大きく損なっていました。

私は生徒側の意見を学内アンケートで集約し、その問題点を**具体的なデータと共に教員に提示**。教員側が抱える懸念点をヒアリングし、**双方の意見の共有を図るための対話の場**を設けました。

残念ながら私の任期中の改善は叶いませんでしたが、この活動がきっかけで、翌年以降、規制緩和に向けた前向きな検討が開始されました。

この経験から、一方の立場（利用者と管理者）だけで互いの意見や背景を想像で補完することの限界を痛感しました。環境向上のためには、まず**事実に基づき問題提起し、当事者間で懸念を共有し、地道に相談と実行を重ねること**が不可欠だと学びました。
